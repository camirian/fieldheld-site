First Public Artifact

Fieldheld Recorder is a small synthetic CLI that creates evidence bundles and policy-gate reports for replay, review, and bounded rollback.

The public repo includes the demo, quickstart, Artifact Walkthrough, Agent Run Record Spec, run review checklist, and sample packets for accepted, rejected, escalated, and blocked runs. It can also create a public-safe review bundle from GitHub PR metadata. It is not a live agent adapter yet. Review Evidence Gate . Open Reviewer Packet Index . Inspect Blocked Run . Current Boundary

Fieldheld starts with synthetic and public-safe examples so autonomous work can be inspected before it is trusted. Production safety, certification, compliance approval, customer adoption, live agent integration, and broader rollback remain outside the current public artifact.
