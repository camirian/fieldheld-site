# Security Policy

This public repository contains a static HTML/CSS site. It has no forms,
analytics, tracking code, or server-side runtime.

## Reporting

Please report suspected security issues privately to the repository owner through
GitHub. Do not open public issues containing secrets, credentials, private
network details, or exploit instructions.

## Public Data Boundary

Reports should not include private keys, API tokens, `.env` contents, customer
data, or machine-specific network identifiers.
